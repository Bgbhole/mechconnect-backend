# MechConnect Backend

This repository contains the backend implementation of **MechConnect**, a vehicle service booking platform that connects customers with nearby mechanics.  
The backend is built using **Spring Boot** and follows a clean, layered architecture with RESTful APIs.

---

## 🚀 Project Overview

The backend handles:
- User registration and authentication
- Role-based workflows for customers and mechanics
- Vehicle service booking and request management
- Business logic, persistence, and API exposure

It is designed to be modular, scalable, and easy to integrate with any frontend client.

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- RESTful APIs
- Maven
